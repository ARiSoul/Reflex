# Reflex

Reflex is a fast-paced, skill-based casual game project focused on
real, honest gameplay inspired by popular playable ads — without the bait.

## Goals
- Short, satisfying game loops (30–60s)
- Skill-based mechanics, no fake gameplay
- Mobile-first, built with Unity and C#
- Clean separation between game logic and engine

## Current Mechanic
- Shoot-the-right-target (additive & multiplier scoring)

## Tech
- Unity (2D)
- C#
- Engine-agnostic Core logic

## Status
Early prototype
